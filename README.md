Project Overview
The objective of this project is to apply deep learning techniques to perform semantic segmentation of brain tumors using Magnetic Resonance Imaging (MRI) data from the public BraTS 2020 (Brain Tumor Segmentation) dataset. Implementing a customized U-Net architecture in PyTorch, the pipeline processes slice files, handles clinical class imbalance via a joint BCE-Dice loss, and evaluates model generalization under different hyperparameter configurations.
